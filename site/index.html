devcodes v1.0.0

A faster, modern alternative to Axios. Built on the Fetch API with full TypeScript support and zero runtime dependencies.

✓ Full TypeScript support with strict types
✓ Request & response interceptors
✓ Timeout + AbortSignal support
✓ Works in Node.js 14+ and all modern browsers
✓ Zero runtime dependencies
✓ Drop-in Axios replacement for most use cases
typescript 
import devcodes, { create } from 'devcodes';

// Quick GET request
const { data } = await devcodes.get('/api/users');

// Custom instance
const client = create({
  baseURL: 'https://api.example.com',
  timeout: 10_000,
  headers: { Authorization: `Bearer ${token}` },
});

// Interceptors
client.requestInterceptors.use((cfg) => {
  cfg.headers!['X-Request-ID'] = crypto.randomUUID();
  return cfg;
});
